HTML Terms

HTML
Hyper Text Markup Language used to design webpages
HTML element
An HTML element is defined by a start tag, some content, and an end tag
HTML tag
An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document
Angle bracket
Angle blacket open and closed to represent tags
HTML attribute
Attribute gives the special value to the element
HTML Comment
Comment used to identify the tags what we used
Html object element
defines a container for an external resource. The external resource can be a web page, a picture, a media player, or a plug-in application.
HTML special character
typically those that can't be easily typed into a keyboard or may cause display issues if typed or pasted into a web page.
